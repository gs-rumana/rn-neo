# RN Neo Docs

Documentation site for [RN Neo](https://github.com/gs-rumana/rn-neo) — an
open-source Neubrutalist UI library for React Native. Built with React Router +
Fumadocs.

## Development

Run the dev server:

```bash
yarn dev
```

## Build

```bash
yarn build
```

Output is generated at `build/client/`. Preview the production build with:

```bash
yarn start
```

## Content

Docs live in `content/docs/` as `.mdx`. Sidebar order is controlled by
`meta.json` files. Theme tokens and Neubrutalist styling live in `app/app.css`.
